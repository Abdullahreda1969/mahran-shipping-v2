# 🏷️ دليل استخدام Labels في مشروع Mahran

## نظام التصنيف:

### حسب النوع:
- 🐛 `bug` - خطأ في النظام
- ✨ `feature` - ميزة جديدة
- 📖 `documentation` - تحديث التوثيق
- 🔧 `enhancement` - تحسين ميزة موجودة

### حسب الأولوية:
- 🚨 `critical` - عاجل/مهم للغاية
- 🔺 `high` - مهم
- 🟡 `medium` - متوسط الأهمية

### حسب المسؤول:
- 🎨 `designer` - مهام المصمم
- 💻 `frontend-dev` - مهام مطور الواجهة
- ⚙️ `backend-dev` - مهام مطور الخلفية

## أمثلة:
- ميزة جديدة في الواجهة: `feature` + `frontend` + `high`
- خطأ في الخلفية: `bug` + `backend` + `critical`
- تحديث التوثيق: `documentation` + `medium`
